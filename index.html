键盘的安全风险及防御措施

信息安全作品赛

一. 摘要

键盘是计算机系统主要的信息输入设备，在生活中的使用可谓十分普遍，但由于其工作原理具有一定的缺陷性，即键盘中的电子元件在运作时会将具有特征的电磁波发射出来，如果运用技术手段对其进行捕获和分析，很有可能会造成按键信息的泄露，因此存在较高的安全风险。本作品针对这一问题，进一步探究并实现了通过键盘泄露的电磁噪声还原按键信息的过程，并设计出相应的防御措施，对传统键盘进行创新性改良。通过对比传统键盘和改良键盘面对电磁泄漏攻击，改良键盘具有以下特点：在键盘电磁泄漏的传播过程中形成很大的无规律噪声干扰，或是在电磁泄漏产生过程中人为产生可控的电磁噪声干扰，形成加密效果，从而使得电磁辐射的分析难度增大。经过多次实验，改良键盘能够抵御多种电磁泄漏攻击，具有良好的安全性和实用性。

二. Attack

(一) 原理

1. 电子设备电磁泄漏

根据麦克斯韦的电磁场理论，变化的电场会激发产生变化的磁场，变化的磁场又会激发产生变化的电场，二者相互激发，产生可以传播的电磁波。因此在电子信息设备中，只要存在时变电流，就会产生电磁波，并且辐射强度与电流强度以及电流变化率正相关。在电流变化波形中，越陡峭的上升沿就会辐射出越强的电磁波。

在电磁泄露的研究中，携带敏感明文信息的电信号称为“红信号”；反之，不携带有价值信息或是携带加密信息的称为“黑信号”。因此，红信号电磁辐射的泄露成为研究的重点，其中红信号的类型主要包括基带信号、调制信号和脉冲信号。

如今大多数电子信息设备使用的都是数字信号，使用二进制表示电路的通断两种状态，因此会经常有上升沿或下降沿的出现，这就伴随着电磁波的产生，发生电磁泄漏现象。

2. 键盘电磁泄漏原理分析

键盘作为常用的电子信息设备，数字信号是其主要信息交互方式，那么经常出现的上升沿和下降沿就必然导致电磁信号泄露现象的出现。

2.1 键盘结构分析

当今键盘主要分为薄膜键盘、机械键盘和静电容键盘，其中薄膜键盘由于其价格便宜、性能较好的优点，生活中所使用的大部分都是这类键盘。它的结构分为键盘外壳、导电薄膜以及控制电路部分。虽然不同类型键盘的结构和触发原理可能有所不同，例如薄膜键盘的触发方式是通过导电薄膜的导电触点压在电路板上导通电路，而电容式键盘是当导电介质之间的电容达到一定程度就导通电路，但键盘最为核心的部分——即控制电路部分的工作原理基本是一致的，分析控制电路数字信息的产生是弄清电磁信号特征性的关键。

2.2 键盘控制电路

键盘控制电路是整个键盘最为核心的部分，主要担任按键扫描识别、编码和传输接口工作；它之所以能将各个按键所表示的数字或字母转换成计算机可以识别的信号，是因为其内部有一组排列成矩阵方式的按键开关，每当有按键被按下，对应的矩阵电路就被接通，进而可以确定按键的位置。为了查询是否有按键被按动，需要运用硬件或软件的方法对矩阵的行列轮转地进行扫描，也称之为行列扫描法。具体的原理是，当没有按键按下时，矩阵的每一行和每一列都被置成高电平，若有按键被按下，则其所在的行列会变成低电平，微处理器会对行列电平进行循环扫描，进而可以确定按下的按键所在位置。当微处理器检测到按键被按动时，会对行列矩阵中其所在的位置进行编码，称之为扫描码；其中按下过程产生的称为接通扫描码（mark），松开按键过程产生的叫做断开扫描码（break）。除了矩阵开关和微处理器，电路中还有晶振以及译码器。其中晶振为键盘中的时钟提供频率基准，译码器将按键的编码信息编译成对应的编码，产生接通扫描码和断开扫描码。

2.3 键盘信息传输过程的电磁泄露

通过上述的键盘工作原理，我们可以分析出完整的键盘信息生成并传输的流程，并判断那些过程会产生红信号泄露。

（1） 键盘的微处理器会对电路行列矩阵进行轮询扫描，检测到低电平时，即可确定有按键被按下，并得到其位置；

（2） 按键被按下和被松开时，还会进行消抖处理，译码器将准确的按键信息编译得到接通扫描码和断开扫描码；

（3） 生成的扫描码将被封装处理，并通过接口电路进行传输。

在这个过程中，接通扫描码和断开扫描码的传输会伴随着数字信号电平的快速变化，因此必然会存在电磁泄漏现象，由于按键扫描码的唯一性和对应性，如果能够截获并分析扫描码生成的电磁红信号，那么便有可能反向推导出按键信息。

4.代码展示

%查找并预测，攻击用
clc;
adc = csvread('1.csv');
Fill=ones(1,3000)*adc(1,1);   %填充
adc=[Fill,adc];               %合并
Hd = lowpass_filter;
adc = filter(Hd, adc);        %调用滤波器
adc=adc(4000:length(adc));

fs=2302;N=length(adc);
n=0:N-1;t=n/fs;
f=n*fs/N;

hold on
plot(t, adc);
counter = 0;
wLength = 550;
colorTable=['b','g','r','c','m','y','k'];  %颜色表
keyTable=['h','n','u','c','s','e'];        %按键表
indexStart = zeros(1,100);                 %区间索引
indexEnd = zeros(1,100);

adcDiff = diff(adc);
adcLength = length(adc);

%根据实际情况调整
%------------------------------
slope = 0;      %斜率阈值
step = 300;     %不同按键之间的间隔长度
rangeMin = 3;   %变化幅度的最大最小值(开区间)
rangeMax = 30;
edgeEL = 100;    %左右扩张范围
edgeER = 500;
%------------------------------

for i = 1:adcLength-1
    if counter>0
        if i<indexEnd(1,counter)+step    %前一个下降沿后一个步长以内不检测
            continue;
        end
    end
    if adcDiff(i)<slope  %提取下降沿
        j = i;
        while j>1 && adcDiff(j)<-0.01
            j=j-1;
        end
        startP = j;
        j = i;
        while adcDiff(j)<0 && j<length(adcDiff)
            j=j+1;
        end
        endP = j;
        rangeD = adc(1,startP) - adc(1,endP);
        if (rangeD > rangeMin && rangeD<rangeMax)   %判断极差是否在范围内
            if(startP-edgeEL>0 && endP+edgeER<adcLength)  %判断是否在边缘    
                    startP = startP - edgeEL;%左右扩张
                    endP = endP + edgeER;
                    counter = counter + 1;
                    indexStart(1,counter) = startP;
                    indexEnd(1,counter)= endP;
            end
        end
    end
end

fprintf("%d个结果\n",counter);
Md1 = loadCompactModel('Model1.mat');   %加载模型
ProbsAll = zeros(counter,length(keyTable));
for j = 1:counter
    plot(t(indexStart(1,j):indexEnd(1,j)),adc(indexStart(1,j):indexEnd(1,j)),colorTable(mod(j,7)+1));
    data=getFeature(adc(indexStart(1,j):indexEnd(1,j)));   %计算特征
    [key_label,Probs] = predict(Md1, data);
    ProbsAll(j,1:length(keyTable)) = Probs;
    fprintf('%c',keyTable(key_label));  %预测结果
end

Defense

展示

通过对键盘电磁泄漏攻击过程的分析可知，电磁辐射主要是在按键按下以及松开过程产生，并通过人体作为导体耦合传播出去，我们可以针对电磁辐射产生和传播的两个过程设计出应对TEMPEST攻击的防御措施。

在键盘电磁泄漏的传播过程中形成很大的无规律噪声干扰，或是在电磁泄漏产生过程中人为产生可控的电磁噪声干扰，形成加密效果，从而使得电磁辐射的分析难度增大。
